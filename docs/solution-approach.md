# Suggested Solution Approach

## 1. Data Collection
Use 2–3 sources:
- Amazon trends
- Google Trends
- Social media
- FDA databases

## 2. Scoring Model
Rank trends using:
- Recency
- Growth rate
- Competition level
- Category relevance

## 3. Filtering Layer
Apply:
- Shelf life constraint
- FDA restrictions
- Tariff risks

Remove non-viable options early.

## 4. Output Layer
Display:
- Top trends
- Reasoning
- Recommended actions

## Key Strategy
Focus on:
> One category deeply rather than everything shallowly
